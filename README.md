# 音标有谱

一个英语音标点读练习页。首页展示 48 个音标，点击即可播放发音。

## 功能

- 48 个英语音标：按单元音、双元音、辅音分组展示。
- 分类筛选：支持按元音、辅音、长短音、发音方式、清浊筛选。
- 点击播放：点击音标播放对应发音，切换时自动停止上一段。
